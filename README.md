# 🚀 EstudaComigo.dev

Landing page desenvolvida para apresentar a proposta da comunidade **EstudaComigo.dev**, uma iniciativa que busca conectar desenvolvedores e estudantes de programação para aprender de forma colaborativa.

O projeto foi desenvolvido utilizando apenas **HTML5**, **CSS3** e **JavaScript**, aplicando conceitos de estrutura semântica, estilização responsiva, manipulação do DOM e consumo de APIs.

## ✨ Funcionalidades

- Landing page moderna e responsiva.
- Navegação entre seções da página.
- Formulário de cadastro.
- Consulta de CEP utilizando a API ViaCEP.
- Preenchimento automático de cidade e estado.
- Modal de confirmação de cadastro.
- Modal informando funcionalidades em desenvolvimento.
- Interface intuitiva e foco na experiência do usuário.

## 🚀 Como executar o projeto

Clone o repositório:

```bash
git clone https://github.com/kauaamelo/estudacomigo-dev.git
```

Acesse a pasta do projeto:

```bash
cd estudacomigo-dev
```

Abra o arquivo `index.html` no navegador ou utilize uma extensão como **Live Server** no Visual Studio Code.
